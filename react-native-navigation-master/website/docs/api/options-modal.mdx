```js
const options = {
  swipeToDismiss
};
```

## `swipeToDismiss`

Control wether this modal should be dismiss using swipe gesture when the `modalPresentationStyle` is `pageSheet`

| Type                                  | Required | Platform |  Default |
| ------------------------------------- | -------- | -------- | -------- |
| boolean                               | No       | Both     | true     |
