### `color`

Set the background color.

| Type  | Required | Platform |
| ----- | -------- | -------- |
| Color | No       | Both     |

### `width`

Set the background width.

| Type   | Required | Platform |
| ------ | -------- | -------- |
| number | No       | Both     |

### `height`

Set the background height.

| Type   | Required | Platform |
| ------ | -------- | -------- |
| number | No       | Both     |

### `cornerRadius`

Set the background cornerRadius.

| Type   | Required | Platform |
| ------ | -------- | -------- |
| number | No       | Both     |
