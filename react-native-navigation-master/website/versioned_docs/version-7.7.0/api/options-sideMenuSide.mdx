title: Side Menu Side Options
sidebar_label: Side Menu Side
---

```js
const options = {
  sideMenu: {
    left: {
      visible: false,
      enabled: true
    }
  }
};
```

### `visible`

Show or hide the side menu.

| Type    | Required | Platform |
| ------- | -------- | -------- |
| boolean | No       | Both     |

### `enabled`

Enable or disable the side menu.

| Type    | Required | Platform |
| ------- | -------- | -------- |
| boolean | No       | Both     |

### `width`

Set the width of the side menu.

| Type   | Required | Platform |
| ------ | -------- | -------- |
| number | No       | Both     |

### `height`

Set the height of the side menu.

| Type   | Required | Platform |
| ------ | -------- | -------- |
| number | No       | Both     |
